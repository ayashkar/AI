# StyleHub Clothes Store (Starter Project)

This is a simple starter web app for a clothing store.

## Features
- Product listing
- Category filter
- Add/remove cart items
- Total cart price calculation
- Simple checkout action

## Run locally
Because this is plain HTML/CSS/JS, you can open `index.html` directly in your browser.

Or run a local server:

```bash
python3 -m http.server 8000
```

Then open: `http://localhost:8000`
